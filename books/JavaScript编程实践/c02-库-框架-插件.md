  * 插件评估十项检查
    * 自己写是不是更好
    * 文档做的好不好
    * 是不是有良好的支持历史
    * 是不是使用标准的参数签名模式
    * 是不是有简单的 HTML 标记要求
    * 所用的 CSS 品质如何
    * 有没有所支持的浏览器的列表
    * 有没有缩小版
    * 有其他人在用吗
    * 能用吗
