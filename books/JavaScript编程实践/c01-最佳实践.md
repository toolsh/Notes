* 松耦合

* JavaScript MVC和模板

  * 模型   -> 视图   （更新）
  * 视图   -> 用户   （看到）
  * 用户   -> 控制器 （使用）
  * 控制器 -> 模型   （更新）

    * 模型：驱动程序的数据
    * 视图：用户界面，使用模板可以在视图中展示模型更容易
    * 控制器：在模型和视图之间传递数据
